Progress Evaluation System is an evaluation website which has 3 primary functionality.It helps you submit progress, evaluate it and generate report for the same.

Q.What does PES mean?

A.It stands for progress evaluation system.

Q.Who all can login to this website?

A.Teachers and students can login by using there USN as User-ID and password.

Q.The website fails to load.What should I do?

A.There might be bandwidth outage .Wait for sometime and try to login again.

Q.I want to search for a particular project, Will I be able to do that?

A.yes! We have a search box on dashboard,you can search for the projects and related information.

Q.Which browser/computer equipment do I need?

A.You will need a reasonably reliable Internet connection to access. All modern computers should work fine on our platform, although we recommend at least 1 GB of memory/RAM. We support the following web browsers:chrome,Firefox.

Q.How can I keep track of deadlines?

A.To help you remember when the deadlines are, we have messaging feature which will help teacher to send you a reminder of projects deadline.

Q.Can I see guide name for the projects?

A.yes,you can see the guide name of each project.
