Progress Evaluation System is an evaluation website which has 3 primary functionality.It helps you submit progress, evaluate it and generate report for the same.

  1. PES page has a home page which has a link to the login page in the top right of the page.

  2. And in the home page you can see the list of ongoing projects and progress of those projects.

  3. Again in the user there are three category,

  1.          1. Admin
  2.          2. Lecturer
  3.          3. Student

  4. In the login page user can enter their respective id and the password and login to their respective pages.

  5. Admin can add the users, assign the projects to the students, and delete the users.

  6. Student can upload the project, search for the projects, send message to the lecturers.

  7. Lecturer can evaluate the project, search for the projects, send message to the students, and comment on the projects.

  8. After session is over all user can logout the session.
